How to Start Services Locally
Since make and Docker aren't in your PATH, use these PowerShell commands:

Terminal 1 — Backend:

powershell
cd api
$env:DATABASE_URL="postgresql://postgres:amantya@123@db.octddaqlkwpmyufouqlj.supabase.co:5432/postgres"
$env:JWT_SECRET="your_jwt_secret_here"
$env:AUTO_MIGRATE="0"
$env:DEV_ALLOW_ALL_CORS="1"
go run ./cmd/api/main.go
Terminal 2 — Frontend:

powershell
cd web
$env:API_SERVER_BASE_URL="http://localhost:8080"
$env:NEXT_PUBLIC_API_BASE_URL="http://localhost:8080"
npm run dev
